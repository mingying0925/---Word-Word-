# SkillBridge 启动方式（Windows 服务器）

## 1. 启动 MySQL

```powershell
net start MySQL
```

MySQL root 密码：通过环境变量 `MYSQL_ROOT_PASSWORD` 设置（3306 端口）

## 2. 启动 Flask（Word 导出微服务）

```powershell
cd D:\SkillBridge\skillbridge-export
C:\Users\cxkj\AppData\Local\Programs\Python\Python313\python.exe -m waitress --host=127.0.0.1 --port=5000 --threads=2 app:app
```

或开发模式：

```powershell
cd D:\SkillBridge\skillbridge-export
python.exe app.py
```

## 3. 启动 Java（主服务）

**生产模式（使用 MySQL）：**

```powershell
cd D:\SkillBridge\skillbridge-web
D:\jdk21\bin\java.exe -jar target\skillbridge-web-0.0.1-SNAPSHOT.jar --server.port=8088 --spring.profiles.active=prod --spring.datasource.url="jdbc:mysql://127.0.0.1:3306/skillbridge?useSSL=false&serverTimezone=Asia/Shanghai&characterEncoding=utf8&allowPublicKeyRetrieval=true" --spring.datasource.username=skillbridge --spring.datasource.password=%DB_PASSWORD% --spring.jpa.hibernate.ddl-auto=update --spring.flyway.enabled=false
```

**开发模式（使用 H2 内存数据库）：**

```powershell
cd D:\SkillBridge\skillbridge-web
D:\jdk21\bin\java.exe -jar target\skillbridge-web-0.0.1-SNAPSHOT.jar --server.port=8080
```

## 4. 验证

| 服务 | 地址 |
|------|------|
| 教师端 | http://localhost:8088/ |
| Flask 健康检查 | http://127.0.0.1:5000/api/health |
| H2 控制台（仅 dev） | http://localhost:8080/h2-console |

默认教师账号：`admin` / 密码通过环境变量 `TEACHER_DEFAULT_PASSWORD` 设置

## 5. 一键启动（需先配置 `deploy\start.bat` 中的路径）

```powershell
D:\SkillBridge\deploy\start.bat
```
