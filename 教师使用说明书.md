## 零、快速上手（5 分钟）

> 本节帮助你用最短时间完成一次完整流程。详细说明请参考后续章节。

**前提**：你已有一份带书签的 `.docx` 模板（书签制作方法见 [Word 导入功能使用说明书](doc/Word导入功能使用说明书.md)）。

```
① 登录  →  ② 创建活动  →  ③ 确认字段  →  ④ 通知学生  →  ⑤ 导出 Word
```

| 步骤 | 操作 | 预期结果 |
|------|------|---------|
| ① 登录 | 打开系统地址 → 点击"教师端" → 输入工号和密码 | 进入仪表盘，看到统计卡片和待办事项 |
| ② 创建活动 | 点击"+ 创建活动" → 填写名称和截止时间 → 上传 `.docx` 模板 → 点击"下一步" | 页面跳转至"确认字段"页，显示解析到的书签列表 |
| ③ 确认字段 | 检查字段列表 → 按需勾选"必填"和"启用" → 点击"确认保存" | 页面跳转回活动列表，顶部显示"活动创建成功"提示 |
| ④ 通知学生 | 复制活动链接发给学生 | 学生访问链接可填写表单 |
| ⑤ 导出 Word | 活动列表 → "查看提交" → 点击"导出 Word"或"一键批量导出" | 浏览器下载填好数据的 Word 文件 |

> **遇到问题？** 请跳转 [常见问题](#八常见问题) 章节排查。

## 一、系统简介

SkillBridge 是一套**职业资格 / 技能等级认定报名自动化系统**。教师只需上传一份带书签的 Word 模板，系统会自动：

1. 解析模板中的可填字段（书签）
2. 生成与原表结构一致的**在线报名页面**
3. 学生在线填写后，**自动填回 Word 模板**并导出

**省去**：
- 手动复印表格
- 学生手写后再人工录入
- 校对格式、字迹、错别字

**V3.1 新增亮点**：
- 表单草稿自动保存（填写中途意外关闭也不丢失）
- 填写进度条与必填标记
- 仪表盘待办事项（即将截止活动、未导入名单提醒）
- 学生可自行下载已提交的 Word 文件
- 模板库管理（一次制作，多次复用）

## 三、核心流程概览

整个报名流程只有 **5 步**：

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  ① 上传模板   │ --> │ ② 确认字段  │ --> │ ③ 导入名单   │ --> │ ④ 学生填写  │ --> │ ⑤ 导出 Word  │
│  Word(.docx)│     │  系统自动    │     │  Excel(可选) │     │  在线表单    │     │  一键导出    │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
```

> 第 ③ 步为可选：如不导入名单，所有学生均可报名；导入名单后，仅名单内学生可提交。

## 四、操作详解

### 4.1 第一步：准备 Word 模板（一次性）

在让学生报名**之前**，你需要准备一份**带书签**的 Word 模板。

#### 什么是书签？

Word 的"书签"就像在文档里**插一根标记针**，告诉系统"这里要填东西"。

#### 如何插入书签？

1. 用 Word 打开你的申报表模板（如"职业技能等级认定申报表.docx"）
2. 找到要学生填写的单元格（如"姓名：______"中的横线）
3. **选中** 那个要填的位置（光标定位到该单元格内）
4. Word 菜单 → `插入` → `书签`
5. 输入书签名（**必须是英文**，如 `name`、`idCard`、`gender`）
6. 点击 `添加`

> **建议书签命名**：`name`（姓名）、`idCard`（身份证号）、`gender`（性别）、`birthday`（出生日期）、`phone`（联系电话）、`address`（地址）、`school`（毕业院校）、`major`（专业）...

#### 如何批量添加书签？

如果模板里要填的位置很多（几十个），一个个手动添加太慢。可以使用项目自带的脚本：

```powershell
# 这个脚本在你的项目 skillbridge-export 目录下
python insert_bookmarks.py
```

具体使用方法参考该脚本说明（`--help`）。

### 4.2 第二步：创建报名活动

1. 在仪表盘或活动列表页，点击 **"+ 创建活动"**
2. 填写：
   - **活动名称**（必填）：如"2026 年春季职业技能等级认定"
   - **截止时间**：学生报名的截止日期（不能选择过去时间）
3. **上传 Word 模板**：点击上传框，选择 `.docx` 文件（不超过 10MB）
   - 或从**模板库**选择已有模板直接创建（可自定义活动名称和截止时间）
4. 点击 **"下一步：解析书签"**

> **预期结果**：页面跳转至"确认字段配置"页，顶部显示"成功解析 N 个书签"。
>
> 创建页底部有**书签制作指南**，可随时查看插入书签的步骤。
> 如果上传失败，已填写的活动名称和截止时间会自动回填，无需重新输入。
> **验证书签是否添加成功**：在 Word 中按 `Ctrl+G` → 选择"书签" → 能看到书签列表说明添加成功。

### 4.3 第三步：确认字段配置

系统会自动解析你模板里的所有书签，展示成列表：

| 字段名（书签） | 显示名 | 必填 | 启用 |
|----------------|--------|------|------|
| name | 姓名 | ✓ | ✓ |
| idCard | 身份证号 | ✓ | ✓ |
| gender | 性别 | ✗ | ✓ |
| ... | ... | ... | ... |

- **显示名**：可修改为学生看到的友好名称（默认为书签名）
- **必填**：勾选后学生必须填写该字段
- **启用**：取消勾选可隐藏不需要的字段（学生填表页不显示）
- 支持**全选/反选**快速操作

确认无误后点击 **"确认保存"**。

> **预期结果**：页面跳转回活动列表，顶部显示绿色"活动创建成功"提示，新活动出现在列表第一行，状态为"报名中"。
>
> 如果字段列表为空，说明 Word 模板里没有书签，保存按钮会自动禁用并显示错误提示。请回到 4.1 重新添加书签。
> 点击"取消"会删除已上传的草稿模板，不影响已发布的活动。

### 4.4 第四步：导入学生名单（可选）

导入名单后，只有名单内的学生可以提交报名（白名单模式）。如不导入，所有学生均可报名。

1. 在活动列表中，点击活动操作栏的 **"学生名单"**
2. 点击 **"下载模板"** 获取 Excel 模板
3. 按格式填写学生信息：

   | 学号 | 姓名 | 身份证号 | 班级（可选） |
   |------|------|----------|-------------|
   | 2026001 | 张三 | 110101200001011234 | 计算机1班 |

4. 点击 **"导入名单"**，选择 `.xlsx` 文件上传（不超过 10MB）

> **预期结果**：导入完成后页面显示"成功导入 N 条名单记录"，活动列表中该活动的"名单人数"列更新。
>
> **Excel 格式要求**：第一行为表头（将被跳过），后续行依次为：学号 | 姓名 | 身份证号 | 班级（可选）。
> 导入时会显示加载动画，请耐心等待。
> 可随时清空名单（已提交的报名记录不会被删除，但学生将不再受白名单限制）。

### 4.5 第五步：通知学生报名

活动发布后，把**活动链接**告诉学生。学生点击链接后：

1. 输入**学号 + 身份证号**登录
2. 看到与原 Word 表格一样的在线表单
3. 填写后点击"提交"

> **提示**：可将 [学生使用说明书](学生使用说明书.md) 一并发给学生，帮助其快速上手。

**学生端的易用性保障**：
- 表单每 30 秒自动保存草稿到浏览器，意外关闭后可恢复
- 提交失败时自动保留已填内容，无需重新填写
- 必填字段有红色 `*` 标记
- 实时显示填写进度条
- 提交前弹出确认对话框，防止误操作
- 移动端键盘自动适配（学号/身份证号输入框）

> 学生只能提交一次，提交后无法自行修改。如发现填写错误，请联系指导教师处理。
> 学生提交后可在查看页下载自己的 Word 申报表。

### 4.6 第六步：查看报名情况

回到教师后台 → 活动列表 → 点击对应活动的 **"查看提交"**

可以看到：
- 活动信息栏（名称、截止时间、状态、提交数）
- 已报名的学生列表（学号、身份证号脱敏显示、提交时间）
- 每个学生的详细数据
- 单个导出 Word 或批量导出 ZIP

**活动列表还显示**：
- 提交人数 / 名单人数
- 即将截止的活动会高亮标记（橙色"即将截止"徽章）
- 支持按名称搜索和按状态筛选

### 4.7 第七步：导出 Word

有三种导出方式：

#### 单个学生导出

在提交列表中，点击某个学生右侧的 **"导出 Word"** 按钮，浏览器会下载一个填好数据的 Word 文件。

> **预期结果**：浏览器下载文件名为`{学生姓名}.docx`，打开后数据已填入对应书签位置，表格样式与原模板一致。

#### 批量导出 ZIP（推荐）

在提交列表页面顶部，点击 **"一键批量导出所有 Word"**，所有学生的 Word 文件会打包成 ZIP 下载。文件名包含学号，方便归档。

> **预期结果**：浏览器下载 ZIP 压缩包，命名为`【{活动名称}】全员申报表汇总.zip`，解压后每个 Word 文件以学生姓名命名。

#### 导出 Excel

点击 **"导出 Excel"** 可下载所有学生提交数据的汇总表格。

> **预期结果**：浏览器下载 `.xlsx` 文件，每行对应一个学生的提交数据。

> 批量导出 ZIP 可能较慢（需逐个解析并填充 Word），页面会显示加载动画，请耐心等待。
> 大量导出可使用**导出任务**功能（见下方），异步处理完成后可下载。
> **导出失败排查**：如 ZIP 中出现`{姓名}_导出失败.txt`，打开查看原因，通常为 Python 导出服务未启动（`.\start.bat status`检查）。

### 4.8 模板库管理

模板库用于保存常用的 Word 模板，创建活动时可直接复用，无需重复上传。

#### 上传模板到库

1. 点击导航栏的 **"模板库"**
2. 在"上传新模板到库"区域，填写模板名称并选择 `.docx` 文件
3. 点击 **"上传到库"**

#### 从活动保存模板

在活动列表中，点击活动操作栏的 **"存为模板"**，输入模板名称即可保存到库。

#### 使用模板创建活动

在模板库列表中，点击 **"用此模板创建活动"**，可自定义活动名称和截止时间。

## 六、教师账号管理

系统支持多教师账号管理。

### 6.1 创建新账号

1. 点击导航栏的 **"账号管理"**
2. 在"创建新账号"区域填写：
   - **工号**：2-20 位字母或数字（如 T001）
   - **姓名**：教师姓名
   - **密码**：至少 6 位
3. 点击 **"创建"**

### 6.2 管理账号

- **禁用/启用**：点击账号右侧的按钮切换状态（禁用后该账号无法登录）
- **重置密码**：在输入框中输入新密码（至少 6 位），点击"重置"

> 工号格式校验：2-20 位字母或数字，不符合格式的工号无法创建。

## 七、日常运维

### 7.1 启动服务

服务器开机后会自动启动服务。如需手动启动：

1. **远程桌面**登录服务器
2. 双击桌面的 `start-skillbridge.bat`（如已设置开机启动）
3. 或打开 PowerShell，执行：
   ```powershell
   cd D:\skillbridge
   .\start.bat
   ```

### 7.2 停止服务

```powershell
cd D:\skillbridge
.\start.bat stop
```

### 7.3 重启服务

```powershell
cd D:\skillbridge
.\start.bat restart
```

### 7.4 查看服务状态

```powershell
cd D:\skillbridge
.\start.bat status
```

会显示：
- Java 主服务：运行中 / 未运行
- Python 导出服务：运行中 / 未运行
- MySQL 数据库：运行中 / 未运行
- 端口监听：8088 / 5000 / 3307

### 7.5 修改教师密码

**方式一：通过账号管理页面（推荐）**

1. 用 admin 账号登录
2. 进入"账号管理"
3. 找到目标教师，在"重置密码"输入框中输入新密码
4. 点击"重置"

**方式二：通过数据库**

```sql
-- 连接到 skillbridge 数据库后执行
UPDATE teachers SET password_hash = '$2a$10$N9qo8uLOickgx2ZMRZoMyeIjZAgcfl7p92ldGxad68LJZdL17lhWy' 
WHERE teacher_id = 'admin';
-- 注意：请使用实际的密码 hash 替换
```

> 想要新密码？找管理员用 `BCryptPasswordEncoder` 工具生成新的 hash。

### 7.6 备份数据

定期备份这两个目录（用 U 盘或共享盘）：

```
D:\skillbridge\uploads\          # 上传的 Word 模板和学生照片
D:\mysql_data\                   # MySQL 数据库（所有数据）
```

## 八、常见问题

### Q1：学生打不开页面
**A**：检查学生是否在校园网，浏览器输入服务器地址看能否打开。如不能，找管理员检查防火墙。

### Q2：教师登录提示"工号或密码不正确"
**A**：默认账号为 `admin`，密码由管理员分配（区分大小写）。如忘记密码，找管理员在账号管理页面重置。

### Q3：上传 Word 模板后字段列表为空
**A**：模板里**没有书签**。用 Word 打开模板 → 选中要填的位置 → 插入 → 书签。创建页面底部有书签制作指南可参考。

### Q4：学生提交后我看不到记录
**A**：
- 学生访问的活动链接是否和你发布的活动一致
- 活动是否已过截止时间
- 教师后台 → 活动列表 → 点击活动 → 查看提交

### Q5：导出的 Word 文件打不开
**A**：检查 Python 导出服务是否启动：`.\start.bat status`，看 Python 是否运行中。

### Q6：学生身份证号校验失败
**A**：系统采用 **GB 11643-1999** 身份证校验码算法。输入的身份证号必须是真实有效的 18 位号码。

### Q7：服务挂了 / 端口被占用
**A**：
```powershell
cd D:\skillbridge
.\start.bat stop      # 停掉
.\start.bat start     # 重启
```

### Q8：怎么彻底重启服务器（应急）
**A**：远程桌面 → 开始菜单 → 重启。重启后服务会自动启动（如果设置了开机自启）。

### Q9：学生填完表后能否修改？
**A**：**不能**。每个身份证号 + 学号只能提交一次。提交前系统会弹出确认对话框提醒学生仔细核对。如确需修改，请联系指导教师处理。

### Q10：怎么添加新的教师账号？
**A**：登录后进入"账号管理"页面，填写工号（2-20 位字母或数字）、姓名、密码（至少 6 位）即可创建。

### Q11：学生填表时意外关闭了浏览器怎么办？
**A**：系统每 30 秒自动保存草稿到浏览器。学生重新打开表单页面时，会看到"已为您恢复上次填写的内容"提示，可继续填写或清除重新开始。

### Q12：怎么知道哪些活动快截止了？
**A**：仪表盘的"待办事项"区块会列出 24 小时内即将截止的活动。活动列表中即将截止的活动也会显示橙色"即将截止"标记。

### Q13：怎么知道哪些活动还没导入名单？
**A**：仪表盘的"待办事项"区块会列出报名中但尚未导入学生名单的活动。

### Q14：批量导出 ZIP 太慢怎么办？
**A**：批量导出大量学生文件时可能较慢，请耐心等待加载动画完成。如需异步处理，可使用"导出任务"功能（导航栏 → 导出任务），完成后可下载。

## 九、术语表

| 术语 | 含义 |
|------|------|
| **书签（Bookmark）** | Word 文档中的定位标记，用于告诉系统"这个位置需要学生填写"。类似在书页里夹一张便签。 |
| **模板** | 教师上传的带书签的 `.docx` 文件，是生成在线表单和导出 Word 的基础。 |
| **活动** | 一次报名流程的完整单元，包含模板、字段配置、学生名单、提交记录等。 |
| **字段** | 模板中一个书签对应的可填写项，如"姓名"、"身份证号"。 |
| **白名单模式** | 导入学生名单后，只有名单内的学生可以提交报名。未导入名单时所有学生均可报名。 |
| **草稿** | 学生填写过程中系统自动保存的临时数据，防止意外关闭导致内容丢失。 |
| **级联删除** | 删除活动时，同时删除该活动下所有学生提交记录和模板文件。 |
| **脱敏显示** | 敏感信息（如身份证号）只显示部分内容，如`110101********1234`。 |
| **JWT** | 系统用于身份验证的安全令牌，存储在浏览器 Cookie 中，2 小时后过期。 |
| **CSRF 防护** | 防止恶意网站伪造请求的安全机制，系统对所有提交操作进行校验。 |
| **TraceId** | 每个请求的唯一追踪编号，便于排查问题。 |
